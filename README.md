The Rebrand mod by kaylendog is a great mod. Sadly kaylendog hasn't made it available for Minecraft versions 1.21.9-1.21.11, even though it works just fine on there. If you wanted to use the mod you had to manually download it from its Modrinth page and couldn't download it in Modpacks or launchers that use tha Modrinth API due to version incompatibility.
Therefore this project lets you download the Rebrand mod for newer version without that kind of effort by setting the compatible versions to 1.21.9-1.21.11 on modrinth and the mod itself.
There are no other changes concerning the mod.
Support for 26.1 is planned, but not available right now, because there are some changes needed for that.

_I'm glad they set the license to GPL v3, otherwise I wouldn't be allowed to do this._

The original readme:

# rebrand

Customize your client brand sent to multiplayer servers.

## Installation

Download the latest release of Rebrand for your client version, and drag it into your `./minecraft/mods` folder.

### Optional Dependencies

Rebrand has support for [Mod Menu](https://www.curseforge.com/minecraft/mc-mods/modmenu), and can be configured via the mod list when this mod is installed.

## Configuration

Rebrand can be configured by editing the `rebrand.toml` file in the `.minecraft/config` folder.

- `enable`: Enables and disables Rebrand. **Default:** `true`
- `brandName`: The brand name to send to the server. **Default**: `vanilla`

When using Mod Menu, the configuration file will be updated when you save the mod's config.
